# B-FlixMovie
Web đặt vé xem phim, đặt bắp nước, giao diện cold color theme, có phân quyền admin/user, build bằng HTML, CSS, JavaScript.

### Người dùng
- Đăng nhập User / Admin.
- Xem danh sách phim hot từ TMDB.
- Tìm kiếm phim.
- Lọc phim theo thể loại.
- Xem chi tiết phim + trailer.
- Đặt vé chọn ghế.
- Đặt bắp nước.
- Xem đơn hàng của mình.
- Copy voucher.
- Sử dụng voucher.

### Admin
- Dashboard thống kê.
- Xem & xử lý tất cả đơn hàng.
- Quản lý voucher.
- Quản lý bắp nước (thêm / sửa / xóa).

## Giao diện
- Dark mode.
- Cold color theme (xanh – tím – cyan).
- Gradient + glassmorphism.
- Responsive mobile / tablet / desktop.

---

## Công nghệ sử dụng
- HTML5.
- CSS3.
- JavaScript.
- Font Awesome.
- TMDB API.
